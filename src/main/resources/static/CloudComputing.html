Cloud Computing

My first experience with cloud computing was with Amazon Web Services.

Here is what I did:

  1. Created my first EC2 instance to run Ubuntu in a t2.micro instance type.

  2. Created a security group to allow SSH connection to my EC2 instance.

  3. Created a security group to allow HTTP (8080) connection to my EC2 instance.

  4. Created a security group to allow HTTPS connection to my load balancer.

  5. Configured my EC2 instance to run Tomcat.

  6. Created a Hosted Zone in Route 53.

  7. Setup my DNS registrar to point to the name servers (NS) associated with the created Hosted Zone.

  8. Created an IAM role with a policy “AmazonEC2ContainerServiceRole” and attached to the EC2 instance.

  9. Created an Application Load Balancer with a Listener to route HTTPS requests to port 8080 on my EC2 instance.

  10. Associated the Load Balancer with the security group allowing HTTPS connection.

  11. Created a new certificate from Amazon Certificate Manager (ACM)

  12. Asociated the certificate with the listener.

  13. Created an Alias in my Hosted Zone, so requests to my domain gets routed to the load balancer.

  14. From a web browser test the configuration by accessing my domain.
